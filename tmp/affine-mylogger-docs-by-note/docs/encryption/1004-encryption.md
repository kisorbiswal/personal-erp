# Note 1004

Tags: #encryption

There are two flavors of encryption algorithms: symmetric and asymmetric. Symmetric algorithms involve the use of a secret key, which is some bit of information that is known only to those participating in the encryption and decryption. With asymmetric encryption, public and private keys are used.
