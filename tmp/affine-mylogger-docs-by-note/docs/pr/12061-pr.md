# Note 12061

Tags: #pr

When taking a new framework, do all the unique actions, e.g validating a ui field, submittings a form, sending data from one page to other, sending data to server, receiving data from server, uploading files. tables, filtered tables, pagination. Do all these before starting the real project and checklist should contain sample code of all these unique actions.
