# Note 45865

Tags: #review

Below are a few guidelines for DEMO. Please setup the valid data accordingly

- [ ] Do not create segments, products or assets with names like Test, 123, personal names etc.,. \n Use proper names you can use names like Samsung F11, Samsung F12, iPhone I2, iPhone I3 etc.,
- [ ] Before taking any action, explain what you are going to do like I’m going to segments list view and click on create button to create a segment
- [ ] Don’t scroll the screen or switch to other screen when someone speaking about some topic in the demo flow. Wait until the completes the discussion or until they show some other screen.
- [ ] If some flow is not working don’t try to troubleshoot during demo. You can say there is some issue and continue with next flow.
- [ ] Demo only happy path scenarios, do not cover all scenarios or validations of the story
