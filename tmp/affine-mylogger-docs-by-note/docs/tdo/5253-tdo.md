# Note 5253

Tags: #tdo

If you are not working with a text book defined Restful backend  - be ready to do a lot of tweaks and overrides to the Backbone.sync. You could technically use $.ajax but there is no clear cut place to encapsulate it, which is why most times using it feels like a hack.
