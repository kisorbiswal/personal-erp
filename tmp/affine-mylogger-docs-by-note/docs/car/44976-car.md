# Note 44976

Tags: #car

Buy soft sponge to clean the car, use cotton wool to clean the sensors and cameras. 
Use soapy water, natural soap.
Put ice cube on chewing gum to solidify it, then remove it as solid.
DON'T use detergent or Alchohol based cleaners. 
Learn to locate the jack correctly and change wheel.

Every 10000 KM/12 months, wheel balancing and swapping should be done.
