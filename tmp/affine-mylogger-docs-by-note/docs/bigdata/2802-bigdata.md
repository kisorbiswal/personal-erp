# Note 2802

Tags: #bigdata

Extracting and loading data with sqoop. When Sqoop is used with hadoop it does the Extract and Load part, MapReduce does the Transform part. Sqoop can be called from traditional ETL tools, essentially allowing Hadoop to be treated as another source or traget.
